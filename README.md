# Consignado WL - Landing Page

## 📋 Sobre o Projeto

Landing page de alta conversão para crédito consignado, desenvolvida com foco em:
- ✅ Design profissional e moderno
- ✅ Conversão via WhatsApp
- ✅ 100% responsiva (mobile-first)
- ✅ Carregamento rápido
- ✅ Animações suaves e profissionais

## 🎨 Estrutura da Página

1. **Hero Section** - Headline forte com CTA principal
2. **Benefícios** - 4 cards destacando vantagens
3. **Público-Alvo** - Segmentação clara (INSS, Servidores, Militares)
4. **Confiança** - Elementos de credibilidade
5. **CTA Final** - Chamada para ação reforçada
6. **Botão Flutuante WhatsApp** - Sempre visível

## 📱 Como Alterar o Número do WhatsApp

O número atual configurado é: **31996270082**

Para alterar, edite o arquivo `index.html` e substitua todas as ocorrências de `5531996270082` pelo novo número.

### Formato do número:
- **55** = Código do Brasil
- **31** = DDD
- **996270082** = Número do telefone

**Exemplo:** Para o número (11) 98765-4321, use: `5511987654321`

### Locais onde o número aparece:
1. Header (botão superior)
2. Hero section (CTA principal)
3. Seção de público-alvo (botão central)
4. CTA final (botão grande)
5. Botão flutuante do WhatsApp

**Dica:** Use a função "Localizar e Substituir" do seu editor de código para trocar todas as ocorrências de uma vez.

## 🔧 Personalização

### Alterar Cores
Edite as variáveis CSS no arquivo `style.css`:

```css
:root {
    --primary-color: #E67E22;
    --primary-dark: #CA6510;
    --primary-light: #F39C12;
    /* ... */
}
```

### Alterar Textos
Todos os textos estão no arquivo `index.html` e podem ser editados diretamente.

### Alterar Taxas e Valores
Procure pelos valores no `index.html`:
- Taxa de juros: "1,2%"
- Parcelas: "84x"
- Tempo de liberação: "30 minutos"
